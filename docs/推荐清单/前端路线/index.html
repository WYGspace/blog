前端路线

TODO：该文章等待完善

暂时来说，学院不会继续讲进阶的前端知识，这都需要自行去学习，

我将起步线定在，会写简单的 JS 代码 + 能使用 JQuery。

往下，有 ES6 版本存在（JS 新版本），提供了许多实用的新语法与函数。同时，还有一门扩展语言（JS 基础上扩展），叫做 TypeScript （简称 TS）。这里我会直接建议学习 TypeScript，可以同时了解到 npm、Webpack。

但从事实上来说。TypeScript 是选修的，并不影响学习框架，但 ES6 不是，这就是 JS 本身的内容。

其次，学习框架 Vue，这是一门简单易上手的框架，由它开始了解前端工具链、脚手架。

而 React 有一定难度（相比较 Vue），这是可选的学习，也许当你上手之后，会更喜欢它也说不定。

而为什么要学习框架，因为当前插件、UI 组件都在围绕前端框架定制开发。只有懂得框架的基础使用，才能使用他们。

如 Naive UI、AntD、Nuxt
