Typora 使用问题

当前 Typora 已经到了正式版本，要付费了，当然不付费也能使用，只是出现弹窗提醒。但是付费之后并没有啥加成，价格较高（￥89）。此处推荐使用正式版之前的版本，从大体上是无差别的，免去付费弹窗提醒。
